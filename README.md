## 💻 Projeto

Aplicação de recordação de memórias, onde o usuário poderá adicionar à uma timeline textos, fotos e vídeos de acontecimentos marcantes da sua vida, organizados por mês e ano.

## 📝 Tecnologias

### Web
• Next.js <br />
• TypeScript <br />
• Tailwind CSS <br />
• Axios <br />
• Lucide-react <br />
• Dayjs <br />
• Js-cookie <br /> 
• Jwt-decode

### Server
• Fastify <br />
• TypeScript & tsx <br />
• Prisma <br />
• Axios <br />
• Zod <br />
• Dotenv

### Mobile
• Expo <br />
• React Native <br />
